Method Details

workspace

autosar.workspace([version = 3.0], [patch = 2], [schema = None], [attributes = None], [useDefaultWriters = True)
参数

  • version (str, float) – AUTOSAR version.

  • patch – AUTOSAR patch version (deprecated argument)

  • schema (None, str) – Used to specify exact XML schema name to use instead of inferring it from version parameter.

  • attributes (None, dict) – User-defined dictionary containing context-related data. The Workspace object just stores the value(s) for later.

  • useDefaultWriters (bool) – If False it disables all XML writing classes. Users can then register their own XML writers.

返回类型

Workspace

Factory method for creating instances of Workspace objects.
